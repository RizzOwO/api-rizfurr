## ```INSTALL Dependents```
> npm install api-rizfurr

## ```UNINSTALL Dependents```
> npm uninstall api-rizfurr

## ```EXAMPLE```
*• Lyric*
```js
const rizz = require('rizfurr-api');

rizz.lyric('You Broke Me First')
    .then(result => {
     console.log(result)
});
```
*• Pinterest*
```js
const rizz = require('rizfurr-api');

rizz.pinterest(q)
    .then(result => {
     console.log(result)
});
```
